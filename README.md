## What this project does

Welcome to Bloxburg Script is a PC-oriented script made for Roblox players who want a compact layer of automation around routine in-game actions. It blends aiming help with collection automation and toggleable controls, so you can turn features on or off without managing an oversized settings list.

The focus here is on keeping the runtime footprint small while still giving you practical control over how the script behaves in a session. With adjustable collection radius values, smooth aiming logic, and reusable hotkey mappings, the script is meant to be straightforward to adapt across different play styles and update cycles.

## Script Features

- Aimbot module for aim assistance in supported gameplay scenarios
- Auto-collect function for streamlining item or resource pickup
- Toggle controls for enabling and disabling features during play
- Smooth aiming behavior for less abrupt movement handling
- Customizable collection radius to adjust pickup range
- Hotkey customization for personal control preferences
- Session persistence for keeping selected settings active during use
- Lightweight execution designed to stay compact while running
